Images Overview

Because images are static web objects that do not provide any inherent elements that are perceivable by screenreaders and other assistive technologies, further steps need to be taken to insure the accessibility of image content.


Alternative textLevel A

A graphic representing an image with alternative text.

Images must have proper alternative text that is an equivalent to the information being conveyed by the image.

Images that are not informative or provide content that is decorative or already conveyed in text do not use alt text. These elements can also be integrated as CSS backgrounds

If the image is complex, provide an alternative explanation in it's context or on a separate page.
