Input Overview

Because interactions with computers and websites can be complex, input should always work as expected.


Pointer Gestures Level A

If your site uses multipoint or path-based gestures,and these are not essential to the site's functionality, then provide a button or link to activate these functions.


LabelLevel A

If an interface element has text, the accessible name/label for the element should include the visible text.


Target SizeLevel A

Clickable elements are no smaller than 44 by 44 pixels. This can be ignored if you provide an alternative target, the target is inline (such as text), it's not author-modified (like a checkbox), or the target's size is important for its functionality.
