Hover Content Overview

While using the web, some content is accessible through hovering over or focusing on an element. This behavior can become inaccessible if not properly designed. This page covers proper usage of hover interaction when displaying content.


Content Displayed on Hover or FocusLevel AAA

A graphic representing a user hovering over a button to reveal content.

When content is shown on hover or focus:

  • The revealed content can be dismissed without moving the pointer or keyboard focus (usually using the ESC key), unless the content represents an input error or does not cover any other page content.
  • Moving the pointer to the revealed content does not make it dissapear
  • The content should remain on the page until the pointer or keyboard focus is moved away from the trigger point, the content is dismissed, or the content is no longer needed.
